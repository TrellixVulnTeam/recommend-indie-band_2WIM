How it works



Last.fm is a music website with information on a ton of musical artists. Each artist gets tags assigned to them by users of the sight. Luckily, Last.fm is nice enough to provide an API so developers like myself can get to the information.


This website takes the five artists you've given, get their tags from Last.fm, then does some calculations to see find similar artists. This is not the same as Last.fm's similar artists feature. This is my own algorithm and should produce different results. On the results page, the percentages indicate why the particular artist was chosen. For example, if you see 'shoegaze 100%', then both artists had the same level of shoegaze on their tags.



If you want to check out the code behind this, it's all open source on Github. Feel free to make a pull request if you're a dev!
