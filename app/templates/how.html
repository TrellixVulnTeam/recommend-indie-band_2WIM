How it works



This website utilizes last.fm, which is a music website conttaining a ton of data about artists. Each artist gets tags assigned to them by users of the site. Luckily, Last.fm is nice enough to provide an API so developers like myself can get to the information.


This website takes the five artists you've given, gets their tags from Last.fm, then does some calculations to find similar artists. This is not the same as Last.fm's similar artists feature. This is my own algorithm and should produce different results. On the results page, the percentages indicate why the particular artist was chosen. For example, if you see 'shoegaze 100%', then the scaled sum of your given artists and the results artist had the same level of shoegaze.
